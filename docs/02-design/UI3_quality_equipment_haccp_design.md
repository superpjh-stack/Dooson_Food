**관련 Plan**: PM2 (품질·식품안전), PM3 (설비·AI·KPI)

---

## 1. 설계 범위

- 통합 품질 대시보드
- F-value 실시간 차트 + AI 예측 알림
- X-Ray 검사 모니터링 + 이물 분류 결과 + 이미지 뷰어
- 설비 실시간 상태 모니터링 + OEE 차트 + 예측 정비 알림 + 모바일 정비 일지
- HACCP 모바일 점검 앱 + CCP IoT 모니터링 + RAG 법규 검색 챗봇 + 자동 리포트

## 2. 디자인 원칙

- **실시간 모니터링 우선**: 차트는 1초 단위 갱신 (WebSocket)
- **현장 신속성**: 알림 → 액션까지 3 클릭 이내
- **AI 신뢰감**: 예측·분류 결과에 항상 Confidence + 출처 표시
- **현장 디바이스**: 데스크톱(품질팀) + 태블릿(현장) + 스마트폰(점검자) 3개 폼팩터 지원

### 4.2 핵심 UX 패턴
- **이중 차트**: F-value (주) + 온도 (보조) 동일 시간축으로 상관관계 시각화
- **Color Band**: 정상/위험 영역을 차트 배경에 표시
- **예측 점선**: 현재 시점 이후의 LSTM 예측은 점선 + 반투명
- **AI 결과 카드**: Confidence + 권장 조치를 카드 형태로 그룹화
- **실시간 갱신**: WebSocket `/ws/quality/fvalue/{sterilizerId}` 1초 단위

## 15. 알림 UX 패턴 (전 화면 공통)

### 15.1 알림 우선순위
| 우선순위 | UI | 채널 |
|---------|-----|------|
| Critical (CCP 이탈, 설비 고장) | 빨간 모달 + 사이렌 | 푸시+SMS+이메일+슬랙 |
| Warning (AI 사전 예측, 임계 근접) | 노란 토스트 + 알림 센터 | 푸시+슬랙 |
| Info (점검 완료, 작업 완료) | 파란 작은 토스트 | 푸시 |

### 15.2 알림 인지 → 액션 사이클
1. **인지**: 토스트 또는 모달
2. **상세 확인**: 알림 센터 또는 상세 화면 이동
3. **AI 보조 조회**: AI Agent에게 컨텍스트 자동 전달
4. **액션**: 시정조치 발행, 정비 요청, LOT 격리 등
5. **검증**: 액션 결과 확인 후 알림 종료

## 16. 차트/그래프 설계 가이드

### 16.1 차트 타입 매핑
| 데이터 종류 | 추천 차트 | 라이브러리 |
|-----------|---------|----------|
| 시계열 (F-value, 진동, 온도) | Line + Area | Recharts |
| 분포 (이물 종류 비율) | Pie / Donut | Recharts |
| 비교 (라인별 OEE) | Bar (Horizontal) | Recharts |
| Heatmap (시간×요일) | Heatmap | Visx |
| 트렌드 + 예측 | Line + Dashed Line | Recharts (별도 시리즈) |
| 대용량 (수만 포인트) | WebGL Line | ECharts |

### 16.2 차트 디자인 원칙
- **임계값 영역**: Color Band로 정상/위험 표시
- **AI 예측**: 점선 + 반투명 + 라벨 ("Forecast")
- **Confidence**: 예측선 주변 음영(±range) 표시
- **인터랙티브**: Hover 시 정확한 값 + 시각, Click 시 드릴다운
- **접근성**: 스크린리더용 데이터 표 함께 제공
