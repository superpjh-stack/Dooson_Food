## 1. 개요

### 1.1 배경
두손푸드는 설비/AI/KPI 영역에서 다음 문제를 가지고 있음:
- **사후 정비**: 설비 고장 후 대응 → 평균 다운타임 4~8시간
- **분리된 AI 기능**: 생산/품질/HACCP AI가 모듈별로만 동작, 통합 컨텍스트 부재
- **KPI 수기 집계**: 월말 엑셀 정리, 실시간 의사결정 불가

### 1.2 목표
- **예측 정비**: LSTM/AutoEncoder/XGBoost 기반 30분~1일 사전 고장 예측
- **AI Agent 통합 허브**: 생산/품질/HACCP 도메인 Agent 통합 (오케스트레이션)
- **자동 KPI**: 생산성/OEE/품질 지표 실시간 자동 집계 + AI 분석/제언

### 1.3 성공 지표
| 영역 | 지표 | AS-IS | TO-BE | 기한 |
|------|------|-------|-------|------|
| 설비 | 사전 고장 예측률 | 0% | 80% (30분~1일 전) | 2027-Q1 |
| 설비 | 비계획 다운타임 | 8시간/주 | 2시간/주 | 2027-Q2 |
| 설비 | OEE (종합설비효율) | 미측정 | 85%+ | 2027-Q1 |
| AI Agent | 도메인 통합 응답 정확도 | N/A | 90%+ | 2027-Q1 |
| KPI | 집계 자동화율 | 0% | 100% | 2026-Q4 |
| KPI | 의사결정 지연 시간 | 1주 | 실시간 | 2026-Q4 |

## 2. 사용자 페르소나

### Persona E1: 설비팀장 (강민호, 50세)
- **역할**: 설비 정비 계획, 부품 재고 관리, 고장 대응
- **Pain Point**: 갑작스러운 설비 고장으로 라인 정지 → 납기 위협
- **Needs**: 사전 예측 알림, 부품 재고 자동 발주, 정비 이력 분석

### Persona E2: 정비 작업자 (오태석, 38세)
- **역할**: 설비 점검, 정비 작업 수행
- **Pain Point**: 종이 정비 일지, 부품 위치 파악 어려움
- **Needs**: 모바일 정비 일지, AR 기반 부품 위치 안내, 작업 매뉴얼 검색

### Persona AI1: AI Agent 사용자 (전 부서)
- **역할**: 생산/품질/안전 의사결정
- **Pain Point**: 부서 간 데이터 단절, 통합 분석 어려움
- **Needs**: 자연어 질의로 도메인 횡단 답변 ("오늘 살균 라인의 OEE는? CCP 이탈은? 예측 고장은?")

### Persona K1: 경영진/공장장 (대표이사)
- **역할**: 전사 의사결정, 투자 우선순위 결정
- **Pain Point**: 월말 엑셀 보고 → 의사결정 지연
- **Needs**: 실시간 KPI 대시보드, AI 인사이트 자동 제언

## 3. 사용자 스토리

### Epic 1: 설비 예측 정비
- **US-EQ-001** (Must) — 설비팀장으로서, 핵심 설비(살균기, X-Ray, 충진기 등)의 진동/전류/온도 데이터가 IoT로 자동 수집되기를 원한다.
- **US-EQ-002** (Must) — 시스템으로서, LSTM 기반 시계열 예측으로 30분~1일 전 설비 이상을 사전 감지한다.
- **US-EQ-003** (Must) — 시스템으로서, AutoEncoder로 비정형 이상 패턴(처음 보는 이상)을 감지하여 알림한다.
- **US-EQ-004** (Must) — 시스템으로서, XGBoost로 F-value/CCP 기반 살균 공정 이탈을 분류한다.
- **US-EQ-005** (Must) — 설비팀장으로서, 설비별 OEE(가동률 × 성능 × 양품률)를 실시간으로 자동 계산하고 싶다.
- **US-EQ-006** (Must) — 정비 작업자로서, 설비 이상 알림 수신 시 모바일 정비 작업 지시를 받고 작업 결과를 사진과 함께 등록하고 싶다.
- **US-EQ-007** (Should) — 설비팀장으로서, 정비 이력에 기반해 부품 교체 주기를 자동 추천받고 싶다.
- **US-EQ-008** (Should) — 설비팀장으로서, 부품 재고가 임계치 이하면 자동 발주 워크플로우가 실행되기를 원한다.
- **US-EQ-009** (Nice) — 정비 작업자로서, AR 글래스로 부품 위치/매뉴얼을 안내받고 싶다.

### Epic 2: AI Agent 통합 허브
- **US-AI-001** (Must) — 사용자로서, 자연어로 "오늘 A라인 OEE 알려줘"라고 질의하면 즉시 답변과 출처를 받고 싶다.
- **US-AI-002** (Must) — 시스템으로서, 4개 도메인 Agent(생산/품질/HACCP/설비)를 오케스트레이션하여 횡단 질의에 답변한다.
- **US-AI-003** (Must) — 생산 AI Agent로서, 생산 계획 최적화·LOT 추적·이상 감지를 수행한다.
- **US-AI-004** (Must) — 품질 AI Agent로서, CCP 모니터링·X-Ray 결과 분석을 수행한다.
- **US-AI-005** (Must) — HACCP AI Agent로서, RAG 기반 법규 검색·CCP 자동 알림·LOT Traceability를 수행한다.
- **US-AI-006** (Must) — 설비 AI Agent로서, 예측 정비·OEE 분석·정비 우선순위 제언을 수행한다.
- **US-AI-007** (Should) — 사용자로서, AI Agent와의 대화 이력이 보존되어 후속 질문에 컨텍스트가 유지되기를 원한다.
- **US-AI-008** (Should) — 시스템으로서, AI Agent가 답변 시 항상 출처(원본 데이터, 문서, 모델명)를 함께 제공한다.
- **US-AI-009** (Nice) — 사용자로서, 음성으로 AI Agent에 질의하고 답변을 들을 수 있다.

### Epic 3: KPI 대시보드
- **US-KPI-001** (Must) — 공장장으로서, 메인 대시보드에서 생산성/품질/설비/안전 4대 KPI를 실시간으로 보고 싶다.
- **US-KPI-002** (Must) — 시스템으로서, 시간/일/주/월/년 단위로 KPI를 자동 집계하여 트렌드 차트를 제공한다.
- **US-KPI-003** (Must) — 공장장으로서, KPI가 목표치 미달 시 자동 알림과 함께 AI가 원인 분석 결과를 제공하기를 원한다.
- **US-KPI-004** (Must) — 경영진으로서, 월/분기 경영 보고서를 AI가 자동 생성하여 PDF로 받고 싶다.
- **US-KPI-005** (Should) — 공장장으로서, 동일 업종/제품군 벤치마킹 데이터와 비교하고 싶다.
- **US-KPI-006** (Should) — 공장장으로서, "다음 분기 생산량 1.5배 증가 시" 같은 What-if 시뮬레이션을 AI에게 요청하고 싶다.
- **US-KPI-007** (Nice) — 부서장으로서, 개인화된 KPI 위젯을 직접 구성하여 대시보드를 커스터마이즈하고 싶다.

- **온프레미스 옵션**: 영업기밀/식품법규 컴플라이언스 시 자체 호스팅 (Llama 3 / Qwen 2.5)

- 식약처 고시 (HACCP 관련)

## 8. 우선순위 매트릭스 (MoSCoW)

### Must Have (Phase 1, 2026 Q3-Q4)
- 설비 IoT 통합, LSTM 예측, AutoEncoder 이상 감지, OEE 자동 계산
- 모바일 정비 일지
- AI Agent 4종 (생산/품질/HACCP/설비) + Orchestrator
- 자연어 질의 인터페이스, 출처 표시
- 메인 KPI 대시보드 (4대 KPI), 시계열 집계
- 목표 미달 알림 + AI 원인 분석
